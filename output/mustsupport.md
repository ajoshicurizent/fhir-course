## Mustsupport

### Must Support

 The Profile elements consist of Mandatory, Must Support, and Additional Saudi national Requirements elements. The sections below define the Server and Client expectations for processing these elements and illustrate how they are displayed and documented. 

### Must Support Elements

 a. Sender shall populate the must support elements if data is present at their end. b. Receiver shall consume the data if present in the FHIR resource instances. Shall not throw error if data is absent until unless it is a mandatory field.
